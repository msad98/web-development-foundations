Links and Images

You can create hyperlinks using the "a" tag. The "href" attribute specifies the URL of the page the link goes to.

  • Visit the absolute path of https://www.example.com
  • Visit page in new tab
  • Visit the relative path of README file
  • MAILTO: click to mail me
  • SMS: Send me SMS with predefined body
  • TEL: Call me on phone

Images

You can add images using the "img" tag. The "src" attribute specifies the path to the image, and the "alt" attribute provides alternative text for screen readers. It is best practice to always set the width and height attributes to avoid layout shifts.

Tables

You can create tables using the "table" tag. Tables are used to display data in a structured format with rows and columns.
